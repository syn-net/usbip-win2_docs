This is a short tutorial written for my mother and sister on how to access the PE-DESIGN security dongle from your computer. The [PE-DESIGN software][] requires that a particular “card reader & writer” device be connected to your computer before it will open.

I have setup a software driver that allows one to use a USB device over any network. In this particular case, this means even across the Internet! The server and security dongle both are physically located in my bedroom.
