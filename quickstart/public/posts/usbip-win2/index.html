This is a short tutorial written for my mother and sister on how to access the PE-DESIGN security dongle from your computer. The [PE-DESIGN software][] requires that a particular “card reader & writer” device be connected to your computer before it will open.

I have setup a software driver that allows one to use a USB device over any network. In this particular case, this means even across the Internet! The server and security dongle both are physically located in my bedroom.

Your two computers are both clients to the server I have setup on my end. This computer is known as pve3.home and pve3-wifi.home to me.

Usage

PE DESIGN Security Dongle

This is the issue at hand – PE Design requires a special USB box to be present before the software will carry on.

Attaching this device

  • Attach the device before you are ready to open PE Design

Detaching this device

  • Always be sure to detach this device once you are done using PE Design.

  • No body else can use the dongle until you detach this device; or your computer goes to sleep, whichever comes first.

  • If you see this message, this means one of two things:

a) the security dongle is attached you are ready to go!

b) somebody else has checked out this device and you can check with the other users to see if they are using it.

software driver install

The software should always be managed by me; this means installation and uninstalling the driver is my job. In the scenario where I am unavailable, you may take a look at the driver’s README and decide whether or not you are comfortable following the instructions precisely. The two most important things you must follow are:

  • always use the OSSign driver release
  • always create a Restore Point before installing or uninstalling this driver!
